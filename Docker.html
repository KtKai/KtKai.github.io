There are multiple ways to complete this assignment however if you are simply wanting a basic Wordpress container this is how to do that - Make sure to have the most recent and updated version of Docker desktop downloaded. - You can also make sure that everything is updated by opening a Docker terminal and using the ( sudo apt update command ) - after everything is up to date go to the top left of the home screen and search for WordPress. - there are multiple different versions of WordPress choose the one that suites your fancy - MAKE SURE TO DOWNLOAD NOT PULL OR RUN - Click the download icon - It should now be in your containers list from there click the port link - Once you click the port it should give you some prompts starting from the language - Follow the prompts and you're done !
